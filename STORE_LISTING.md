# Chrome Web Store Listing

## Name
Aftermark — Your Bookmarks Are Breadcrumbs

## Summary (132 chars)
Turn forgotten bookmarks into actionable insights. Local-first analysis, smart cleanup, and intent recovery.

## Description

Every bookmark you saved carried an unfinished intention — something to read, compare, learn, decide, or build. Aftermark recovers those intentions and helps you act on them.

**The problem**: You have thousands of bookmarks. Most are forgotten. You can't find what you need. You don't remember why you saved half of them. Your bookmark bar is a graveyard of abandoned ideas.

**Aftermark fixes this.** It imports your entire bookmark collection, classifies every link, detects patterns, and gives you tools to understand and clean up what you've accumulated.

KEY FEATURES

Dashboard & Analytics
- Total bookmarks, duplicates, dead links, stale count, and health score at a glance
- Top domains, content type distribution, bookmarks per month
- Bookmark rate tracking (per day/week/month)

Smart Classification
- Automatically classifies bookmarks into 17 content types: articles, videos, docs, GitHub repos, news, shopping, forums, academic papers, and more
- Domain-level pattern matching for 50+ popular sites
- Favicons and health indicators on every row

Intelligent Cleanup
- 7-step Cleanup Wizard walks you through: duplicates, likely duplicates, dead links, stale bookmarks, single-visit domains, dead folders
- Fuzzy duplicate detection finds similar titles on the same domain
- Health scores (0-100) based on age, revisit history, link status, and classification

Clusters & Sessions
- Automatic grouping by domain, folder, URL path, time session, and GitHub project
- Session reconstruction: see what research sessions looked like based on bookmark timestamps
- Merge, rename, and manage clusters

Full Tag System
- Add custom tags to any bookmark with smart suggestions
- Tag-based filtering and clustering
- Tag Manager: rename, merge, delete, color-code
- Bulk tagging across selections

Timeline & Insights
- Month-by-month sparkline timeline of your bookmarking history
- Personality analysis: are you a Developer, Researcher, Explorer, or Shopper?
- Peak hours, busiest months, longest drought between bookmarks

Real-time Monitoring
- Automatically captures and classifies new bookmarks as you browse
- Live tab refresh when bookmarks change externally
- Badge count on the extension icon

Export
- CSV export of all bookmarks or selections
- Markdown reading lists from any cluster
- HTML bookmark file export (Netscape format)
- Comparison tables for shopping clusters

PRIVACY FIRST
- All data stored locally in your browser (IndexedDB)
- Zero telemetry, zero analytics, zero tracking
- No account required
- No server, no cloud, no backend
- Future AI features will use your own API key (BYOK)

NO AI REQUIRED
Aftermark works entirely offline using deterministic heuristics. All classification, clustering, health scoring, and insights are computed locally. AI enrichment is planned as an optional add-on for users who want deeper analysis.

Built with TypeScript, Chrome MV3, IndexedDB, and zero frameworks.
